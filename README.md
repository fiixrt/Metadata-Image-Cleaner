[Metadata Image Cleaner](https://metadata-image-cleaner.vercel.app) is a utility web app for inspecting and removing metadata from images directly in your browser.

This tool was made for those moments where you want to share a photo but donâ€™t want to expose hidden EXIF data like GPS location, camera model or timestamps.  
**Just upload your image and clean it instantly.**

## Usage

### Development

Just run and visit http://localhost:5173

```bash
npm install
npm run dev
```

## Build

To build the app, run

```bash
npm run build
```

And you will see the generated files inside `dist` ready to be deployed.
